### Start
```
npm install
npm run build:dev
```

### Storybook
```
npm run storybook
```

### Start webserver for development
```
npm run start
npm run server
```

### Build
```
npm run build:dev
npm run build:test
npm run build:staging
npm run build:production
```

### Lint
```
npm run eslint
npm run stylelint
```

### Test
```
npm run test
npm run tests
```
